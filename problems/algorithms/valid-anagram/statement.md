# Valid Anagram

## Description

Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise.

## Input

- `s`: source string
- `t`: target string

## Output

- Boolean result

## Example

Input:

```text
s = "anagram", t = "nagaram"
```

Output:

```text
true
```
