# One Gradient Descent Step

## Description

Given current `weights`, `gradients`, and `learning_rate`, compute updated weights after one gradient descent step.

Formula:

`w_new[i] = w[i] - learning_rate * gradients[i]`

## Input

- `weights`: list of floats
- `gradients`: list of floats
- `learning_rate`: float

## Output

- list of updated weights rounded to 6 decimals
