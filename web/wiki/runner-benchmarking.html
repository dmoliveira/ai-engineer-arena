Runner and Benchmarking

Use local runners for trusted validation before publishing or submitting.

Use runner/run_problem.py for Python and runner/run_shell_problem.py for shell tracks.

Quick commands

python3 runner/run_problem.py --problem-dir problems/algorithms/two-sum-hash --solution examples/solutions/two_sum_hash.py
python3 runner/run_shell_problem.py --problem-dir problems/shell_lab/log-level-counter --command "cut -d' ' -f2 {input_file} | sort | uniq -c | awk '{print $2, $1}'"

Metric labels

  • strict: enforced runtime/test limits
  • advisory: environment-dependent, informational
Browser metrics on GitHub Pages are advisory; CI/local runner metrics should be treated as source of truth.
